# JaxPIP

Jax Implementation of Permutation Invariant Polynomial (PIP) based descriptor.

## Features

Supported descriptors:

- Origin permutation invariant polynomial (PIP) [1,2]
- Many body permutation invariant polynomial (MB-PIP)
- Fragment permutation invariant polynomial (Frag-PIP)
